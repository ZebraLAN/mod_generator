# 更新日志 / Changelog

本文档记录 Stoneshard Mod Generator 的版本更新历史。

---

## [0.9.5] - 2026-01-22

### 修复 / Fixed
- 修复勾选"排除随机生成"选项后，物品仍会在容器/商店中随机出现的问题

### 改进 / Improved
- 属性编辑逻辑优化：默认使用浮点输入（步进 0.1），仅回合数、格数、伤害数值等严格整数属性使用整数输入；部分微调属性（如饱腹变化、口渴变化等）使用更精细的 0.01 步进
- 优化属性编辑器界面布局，改为垂直排列
- 优化分类与子分类选择器，合并为单行流式布局

## [0.9.4] - 2026-01-18

### 修复 / Fixed
- 修复打开容器时可能卡住的问题（将 `scr_loot_from_tables` 的 patch 方式改为字节码汇编替换，避免反编译器错误）

## [0.9.3] - 2026-01-18

### 修复 / Fixed
- 修复 `scr_weapon_array_get_consum` 的 patch 逻辑错误，该问题导致非混合物品绕过 `special` / `special exc` 标签检查，在容器中生成不应出现的物品。

## [0.9.2] - 2026-01-17

### 修复 / Fixed
- 修复 `scr_find_item_candidates` 脚本逻辑中的多项问题：
    - 解决了变量 `_filter_tag` 未定义导致的编译错误。
    - 修正了数据类型转换逻辑中导致编译失败的语法错误。
- 修复混合物品 `Step_0` 事件生成时的条件分支语法错误。

## [0.9.1] - 2026-01-17

初始发布版本。
